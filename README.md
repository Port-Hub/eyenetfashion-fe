## Prototype Website

#### Tech Stack 
    This Project is created with
    * Vite - v3.0.7
    * React - v18.2.0
    * TailwindCSS - v3.1.8

#### Local Setup 
    Local Git Clone
``` 
git clone https://github.com/RahulRST/prototype.git

    Package Installation
```
```
cd prototype
npm install
```
    Local Hosting
```
npm run dev
```
