## Prototype Website

#### Tech Stack 
    This Project is created with
    * Vite - v3.0.7
    * React - v18.2.0
    * TailwindCSS - v3.1.8

#### Local Setup 

* To locally clone the repository

        git clone https://github.com/RahulRST/prototype.git

* To install the required packages

        cd prototype
        npm install

* Local Hosting of React App

        npm run dev
